TimelineJS — tacman fork

A fork of NUKnightLab/TimelineJS3 adding a modern, no-build, JSON-first runtime alongside the original webpack-bundled library — built for Symfony/AssetMapper-friendly integrations.

  • Timeline demo No-build runtime rendering events, eras, media, captions, groups, and overlay chips from a JSON feed
  • Exhibit story demo No-build runtime rendering a curated bookmark folder as ordered blocks with assets and connectors
  • README Overview, modern runtime docs, and links to the original TimelineJS documentation
