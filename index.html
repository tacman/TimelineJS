TimelineJS — tacman fork

A fork of NUKnightLab/TimelineJS3 adding a modern, no-build, JSON-first runtime alongside the original webpack-bundled library — built for Symfony/AssetMapper-friendly integrations. This fork does not support the Knight Lab authoring tool; consuming apps should provide timeline or exhibit-story JSON directly.

  • Timeline demo No-build runtime rendering events, eras, media, captions, groups, and overlay chips from a JSON feed
  • TimelineJS JSON format The timeline runtime consumes this JSON shape directly; it does not use the authoring tool or Google Sheets publishing flow
  • Story contract Framework-neutral JSON/YAML story model for bookmarks, saved searches, blocks, references, and projections
  • Survos Exhibit Bundle Symfony authoring and publishing integration for OpenFoto/zm-backed stories
  • Exhibit story demo No-build runtime rendering a curated bookmark folder as ordered blocks with assets and connectors
  • README Overview, TimelineJS JSON format links, modern runtime docs, and story contract links
